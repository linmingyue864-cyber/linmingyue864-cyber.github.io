Andrenna Music

Nền tảng âm nhạc số

  • ✓
    Upload, stream & phân phối nhạc không giới hạn
  • ✓
    Waveform visualizer + lyrics sync thời gian thực
  • ✓
    Hệ thống follow, playlist & discovery thuật toán AI
  • ✓
    Monetization: Tipping, Pre-save, NFT Drop tích hợp
  • ✓
    API mở cho third-party integration (Spotify, Apple Music)
  • ✓
    Analytics chi tiết: stream, region, demographic
🔄 Tương tự SoundCloud nhưng tập trung vào thị trường Việt Nam & SEA với hỗ trợ thanh toán nội địa

Andrenna Drive

Kho lưu trữ thông minh

Giống Google Drive nhưng được tối ưu cho dữ liệu nhạy cảm, nội dung sáng tạo và workflow cộng tác của các team nhỏ, studio và agency Việt Nam.

  • ✓
    Lưu trữ đám mây mã hóa end-to-end AES-256
  • ✓
    Xem trước file: video, audio, PDF, PSD, Figma ngay trên web
  • ✓
    Chia sẻ link thông minh với quyền hạn chi tiết & expiry
  • ✓
    Tích hợp tự động với Andrenna Music (lưu nhạc = chia sẻ thẳng)
  • ✓
    Version control cho file sáng tạo (track thay đổi, rollback)
  • ✓
    Offline sync desktop & mobile, zero-knowledge encryption tùy chọn
🔄 Tương tự Google Drive nhưng data center đặt tại Việt Nam, tuân thủ Nghị định 13/2023/NĐ-CP về bảo vệ dữ liệu cá nhân
Phân tích kỹ thuật

Kiến trúc SaaS hiện đại

Mỗi sản phẩm được xây dựng trên nền tảng microservice độc lập, chia sẻ infrastructure chung để tối ưu chi phí.

⚡

Microservice Architecture

Mỗi domain (auth, storage, streaming, analytics) là một service độc lập. Scale riêng lẻ, deploy độc lập, fault isolation hoàn toàn.

gRPC Kubernetes Docker Istio
🔐

Zero-Trust Security

Xác thực JWT + OAuth 2.0, mã hóa TLS 1.3 cho mọi kết nối, AES-256 cho dữ liệu tĩnh. Không tin tưởng bất kỳ request nào mặc định.

JWT OAuth 2.0 TLS 1.3 AES-256
📡

CDN & Edge Delivery

Audio/video được phục vụ qua CDN toàn cầu với PoP tại Hà Nội, TP.HCM, Singapore, Tokyo. Latency dưới 30ms cho 95% người dùng SEA.

Cloudflare R2 HLS/DASH Edge Cache
🗃️

Data & Storage Layer

PostgreSQL cho relational data, Redis cho session & cache, MinIO-compatible object storage cho file nhạc và tài liệu. Backup 3-2-1.

PostgreSQL Redis MinIO ClickHouse
🤖

AI / ML Pipeline

Recommendation engine cho Music, auto-tagging & transcription, smart search với vector embeddings, anomaly detection bảo mật tự động.

Whisper AI pgvector PyTorch FAISS
📊

Observability Stack

OpenTelemetry tracing, Prometheus metrics, Grafana dashboard, Loki log aggregation. SLA monitoring real-time với alert đa kênh.

Prometheus Grafana Loki OTel

Roadmap 2025–2026

Music Streaming Auth
Q2 2025 · Hoàn thành ✓
MVP Launch – Andrenna Drive Beta
Upload & lưu trữ file, chia sẻ link, preview video/PDF/audio. Tích hợp Music ↔ Drive: lưu bài nhạc thẳng lên Drive, stream từ Drive link.
