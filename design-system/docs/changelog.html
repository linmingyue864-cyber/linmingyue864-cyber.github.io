v1.0.0 — Ra mắt chính thức Hiện tại
Phiên bản đầu tiên của BHDS — BH Design System. Thiết lập toàn bộ nền tảng thiết kế: token, thành phần cơ bản và tài liệu hướng dẫn tiếng Việt đầy đủ.
MớiHệ thống màu sắc đầy đủ: 9 sắc độ Brand Blue, 10 sắc độ Neutral, 3×3 màu ngữ nghĩa (Success, Warning, Danger, Info)
MớiThang kiểu chữ 11 cấp (10px–40px) với font Inter và JetBrains Mono
MớiHệ thống khoảng cách 16 bậc dựa trên bội số 4px
MớiThư viện Design Tokens đầy đủ dưới dạng CSS Custom Properties
MớiComponent: Button (4 biến thể × 3 kích thước + states)
MớiComponent: Input, Textarea, Select, Checkbox, Radio, Switch
MớiComponent: Badge, Alert (4 biến thể), Progress Bar, Toast Notification
MớiComponent: Card, Avatar, Tag/Chip, Tooltip, Table
MớiThư viện 32 biểu tượng SVG nội tuyến (4 nhóm: điều hướng, hành động, trạng thái, phương tiện)
MớiTài liệu tiếng Việt 11 trang: Giới thiệu, Nguyên tắc, Màu sắc, Kiểu chữ, Khoảng cách, Biểu tượng, Thành phần, Button, Form, Phản hồi, Design Tokens
MớiGiao diện responsive với sidebar thu gọn trên mobile
MớiToast notification system được xây dựng bằng JavaScript thuần
10 tháng 5, 2025
