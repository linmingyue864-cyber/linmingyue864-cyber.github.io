Biểu tượng

BHDS sử dụng biểu tượng SVG nội tuyến — không phụ thuộc vào thư viện ngoài, có thể đổi màu bằng CSS và scale không mất chất lượng. Nhấn vào icon để xem mã SVG.

ℹ️
Tất cả icon được vẽ trên lưới 16×16. Kích thước hiển thị khuyến nghị: 16px, 20px, 24px. Dùng fill="currentColor" để kế thừa màu từ CSS.

Cách sử dụng trong HTML

<!-- Kích thước 20px, màu kế thừa từ CSS -->
<svg width="20" height="20" viewBox="0 0 16 16" fill="currentColor">
  <path d="M8 1.5L1.5 7H3v7.5h4V11h2v3.5h4V7h1.5L8 1.5z"/>
</svg>

<!-- Trong nút -->
<button class="btn btn-primary">
  <svg width="14" height="14" viewBox="0 0 16 16" fill="currentColor">
    <path d="M8 3v10M3 8h10"/>
  </svg>
  Thêm mới
</button>

<!-- Màu tùy chỉnh bằng CSS -->

     style="color: var(--color-brand-500);">
