Design Tokens

Design tokens là các hằng số có tên — đại diện cho mọi quyết định thiết kế trong BHDS. Sử dụng token thay vì giá trị tùy ý để đảm bảo tính nhất quán và dễ bảo trì.

ℹ️
Cách sử dụng: Tất cả token được định nghĩa là CSS Custom Properties trong :root của styles.css. Nhấn vào token bất kỳ để sao chép.

Cách sử dụng trong CSS

/* Đúng — dùng token */
.my-component {
  color: var(--color-neutral-900);
  background: var(--color-brand-100);
  padding: var(--space-400) var(--space-600);
  border-radius: var(--border-radius-xl);
  font-size: var(--font-size-100);
  font-weight: var(--font-weight-medium);
  box-shadow: var(--shadow-sm);
  transition: background var(--transition-normal);
}

/* Sai — giá trị tùy ý không dùng token */
.my-component {
  color: #2C2C2C;        /* ❌ */
  padding: 16px 24px;    /* ❌ */
  border-radius: 8px;    /* ❌ */
}
