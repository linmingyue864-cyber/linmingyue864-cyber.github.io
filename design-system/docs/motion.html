Chuyển động

Chuyển động trong BHDS phục vụ mục đích rõ ràng — không trang trí. Mỗi animation truyền đạt phản hồi, hướng dẫn sự chú ý hoặc làm rõ mối quan hệ không gian.

Nguyên tắc chuyển động

⚡
Nhanh & Có mục đích

Animation micro nên ≤150ms. Animation chuyển trang ≤350ms. Người dùng không nên chờ.

🧭
Hướng dẫn sự chú ý

Chuyển động phải giúp người dùng hiểu điều gì đã thay đổi và tại sao. Không gây mất tập trung.

♿
Tôn trọng prefers-reduced-motion

Luôn kiểm tra prefers-reduced-motion. Cung cấp trải nghiệm không animation cho người dùng nhạy cảm.

Thời lượng

Dùng 3 token thời lượng cơ bản. Không tự đặt giá trị tùy ý.

TokenGiá trịDùng cho
--transition-fast100msHover, focus ring, màu sắc, opacity nhỏ
--transition-normal200msTooltip xuất hiện, switch, trạng thái thành phần
--transition-slow300msSidebar mở, progress bar, chuyển trang nhẹ

Hàm easing

Easing quyết định cảm giác tự nhiên của chuyển động. BHDS ưu tiên cubic-bezier(.22,1,.36,1) cho các thành phần xuất hiện từ trạng thái ẩn.

Code mẫu

/* Sử dụng token thay vì giá trị tùy ý */
.btn {
  transition: background var(--transition-fast),
              color var(--transition-fast),
              box-shadow var(--transition-fast);
}

/* Sidebar slide */
.sidebar {
  transition: transform var(--transition-normal);
}

/* Fade in khi xuất hiện */
@keyframes fade-in {
  from { opacity: 0; transform: translateY(8px); }
  to   { opacity: 1; transform: translateY(0); }
}
.modal { animation: fade-in 200ms cubic-bezier(.22,1,.36,1); }

/* Tôn trọng prefers-reduced-motion */
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: .01ms !important;
    transition-duration: .01ms !important;
  }
}
