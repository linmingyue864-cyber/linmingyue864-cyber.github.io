BHDS CSS Framework

Bộ framework CSS thuần — không phụ thuộc build tool, dùng ngay bằng một thẻ <link>. Bao gồm hệ thống token, utility class, layout grid, và sub-framework theme Adobe.

Không cần build
Nhúng trực tiếp qua CDN hoặc tải file về dự án.
Token-based
Toàn bộ giá trị màu, khoảng cách, font đều qua CSS custom property.
Có theme
Sub-framework adobe.css áp dụng ngôn ngữ thiết kế Adobe.
Dark mode
Hỗ trợ sáng/tối tự động qua data-theme="dark".

Cài đặt

Có hai cách tích hợp BHDS CSS vào dự án của bạn.

Thêm thẻ sau vào <head> của trang:

HTML 
<!-- BHDS CSS — Core -->
<link rel="stylesheet" href="styles.css" />

<!-- Tuỳ chọn: Theme Adobe -->
<link rel="stylesheet" href="adobe.css" />

Tải file về và đặt vào thư mục dự án:

Shell 
# Tải BHDS CSS Core
curl -O https://tiendatweb.vercel.app/design-system/docs/styles.css

# Tải theme Adobe (tuỳ chọn)
curl -O https://tiendatweb.vercel.app/design-system/docs/adobe.css

Cấu trúc file

BHDS CSS được phân tách thành các lớp rõ ràng theo kiến trúc từ nền tảng đến theme.

Text
bhds/
├── styles.css          ← Core framework (tokens + reset + components)
├── adobe.css         ← Sub-framework: theme Adobe
└── themes/
    ├── adobe.css     ← Bản đầy đủ có khai báo @layer
    └── minimal.css   ← Chỉ override token màu và font

Sub-Framework Theme

BHDS CSS hỗ trợ hệ thống theme qua file adobe.css — một lớp ghi đè token được thiết kế theo ngôn ngữ Adobe Spectrum. Tải sau bhds.css để kích hoạt.

bhds.css — Core
Token trung lập
Component cơ bản
Reset CSS
Grid & Layout
Dark mode built-in
adobe.css — Theme
Ghi đè token màu Adobe
Font Source Sans 3 + Source Code Pro
Border radius Adobe (4px / 6px)
Shadow Adobe Spectrum
Biến thể component Adobe

Sử dụng cả hai file:

HTML
<link rel="stylesheet" href="bhds.css" />
<link rel="stylesheet" href="adobe.css" />
Luôn nhúng bhds.css trước adobe.css. Theme file chỉ ghi đè token, không khai báo lại component.

Dark Mode

BHDS CSS tích hợp sẵn dark mode dựa trên thuộc tính data-theme trên thẻ <html>. Không cần thêm file CSS.

HTML
<!-- Light mode (mặc định) -->
<html lang="vi">

<!-- Dark mode -->
<html lang="vi" data-theme="dark">

<!-- Theo hệ điều hành (dùng JS) -->
<html lang="vi" data-theme="auto">

Tuỳ chỉnh Token

Ghi đè CSS custom property trong :root để tạo theme riêng mà không cần sửa source.

CSS
:root {
  /* Thay màu brand */
  --color-brand-500: #7C3AED;
  --color-brand-600: #6D28D9;

  /* Thay font */
  --font-family-base: 'Roboto', sans-serif;

  /* Bo góc ít hơn */
  --border-radius-md: 2px;
  --border-radius-lg: 4px;
}
