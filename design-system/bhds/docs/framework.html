Framework

BHDS CSS Framework

Bộ framework CSS thuần — không phụ thuộc build tool, dùng ngay bằng một thẻ <link>. Bao gồm hệ thống token, utility class, layout grid, và sub-framework theme Adobe.

Không cần build
Nhúng trực tiếp qua CDN hoặc tải file về dự án.
Token-based
Toàn bộ giá trị màu, khoảng cách, font đều qua CSS custom property.
Có theme
Sub-framework adobe.css áp dụng ngôn ngữ thiết kế Adobe.
Dark mode
Hỗ trợ sáng/tối tự động qua data-theme="dark".
Tương thích đa framework
Dùng song song với Bootstrap, Tailwind CSS hoặc bất kỳ framework nào — không xung đột.

Cài đặt

Có hai cách tích hợp BHDS CSS vào dự án của bạn.

Thêm thẻ sau vào <head> của trang:

HTML 
<!-- BHDS CSS — Core -->
<link rel="stylesheet" href="bhds.css" />

<!-- Tuỳ chọn: Theme Adobe -->
<link rel="stylesheet" href="adobe.css" />

Tải file về và đặt vào thư mục dự án:

Shell 
# Tải BHDS CSS Core
curl -O https://bhds.example.com/dist/bhds.css

# Tải theme Adobe (tuỳ chọn)
curl -O https://bhds.example.com/dist/adobe.css

Cấu trúc file

BHDS CSS được phân tách thành các lớp rõ ràng theo kiến trúc từ nền tảng đến theme.

Text
bhds/
├── bhds.css          ← Core framework (tokens + reset + components)
├── adobe.css         ← Sub-framework: theme Adobe
└── themes/
    ├── adobe.css     ← Bản đầy đủ có khai báo @layer
    └── minimal.css   ← Chỉ override token màu và font

Sub-Framework Theme

BHDS CSS hỗ trợ hệ thống theme qua file adobe.css — một lớp ghi đè token được thiết kế theo ngôn ngữ Adobe Spectrum. Tải sau bhds.css để kích hoạt.

bhds.css — Core
Token trung lập
Component cơ bản
Reset CSS
Grid & Layout
Dark mode built-in
adobe.css — Theme
Ghi đè token màu Adobe
Font Source Sans 3 + Source Code Pro
Border radius Adobe (4px / 6px)
Shadow Adobe Spectrum
Biến thể component Adobe

Sử dụng cả hai file:

HTML
<link rel="stylesheet" href="bhds.css" />
<link rel="stylesheet" href="adobe.css" />
Luôn nhúng bhds.css trước adobe.css. Theme file chỉ ghi đè token, không khai báo lại component.

Dark Mode

BHDS CSS tích hợp sẵn dark mode dựa trên thuộc tính data-theme trên thẻ <html>. Không cần thêm file CSS.

HTML
<!-- Light mode (mặc định) -->
<html lang="vi">

<!-- Dark mode -->
<html lang="vi" data-theme="dark">

<!-- Theo hệ điều hành (dùng JS) -->
<html lang="vi" data-theme="auto">

Tuỳ chỉnh Token

Ghi đè CSS custom property trong :root để tạo theme riêng mà không cần sửa source.

CSS
:root {
  /* Thay màu brand */
  --color-brand-500: #7C3AED;
  --color-brand-600: #6D28D9;

  /* Thay font */
  --font-family-base: 'Roboto', sans-serif;

  /* Bo góc ít hơn */
  --border-radius-md: 2px;
  --border-radius-lg: 4px;
}

BHDS JS Framework

Thư viện JavaScript thuần, không phụ thuộc — cung cấp tương tác cho mọi component trong BHDS. Hoạt động hoàn toàn qua thuộc tính data-*, không cần viết thêm JS.

Cài đặt

Thêm script trước thẻ </body>. BHDS JS tự động khởi tạo tất cả component khi DOM sẵn sàng.

HTML
<!-- Đặt trước </body> -->
<script src="bhds.js"></script>

Hoặc import dạng ES Module:

JavaScript
import BHDS from './bhds.js';

// Tự động init tất cả component
BHDS.init();

// Hoặc init riêng từng module
BHDS.initTabs();
BHDS.initModals();
BHDS.initToast();

Data Attributes API

Tất cả component có thể kích hoạt chỉ bằng thuộc tính HTML — không cần viết JavaScript.

Tabs

HTML
<div data-tabs>
  <button data-tab-trigger>Tab 1</button>
  <button data-tab-trigger>Tab 2</button>

  <div data-tab-panel>Nội dung 1</div>
  <div data-tab-panel>Nội dung 2</div>
</div>

Accordion

HTML
<div data-accordion-item>
  <button data-accordion-trigger>Câu hỏi</button>
  <div data-accordion-panel>Nội dung trả lời</div>
</div>

Modal

HTML
<!-- Nút mở -->
<button data-modal-open="myModal">Mở modal</button>

<!-- Modal -->
<div class="modal-backdrop" id="myModal">
  <div class="modal">
    <div class="modal-header">
      <h3>Tiêu đề</h3>
      <button data-modal-close>×</button>
    </div>
    <div class="modal-body">Nội dung</div>
  </div>
</div>

JavaScript API

Dùng API programmatic khi cần điều khiển component từ code.

showToast()

JavaScript
// Hiển thị toast thông báo
showToast('Đã lưu thành công!', 'success');
showToast('Có lỗi xảy ra', 'danger');
showToast('Cảnh báo: token hết hạn', 'warning');
showToast('Đang xử lý...', 'info');

// Tham số: showToast(message, type, duration?)
// type: 'info' | 'success' | 'warning' | 'danger'
// duration: ms (mặc định 3000)

BHDS.openModal() / closeModal()

JavaScript
// Mở modal theo ID
BHDS.openModal('confirmModal');

// Đóng modal theo ID
BHDS.closeModal('confirmModal');

// Đóng tất cả modal đang mở
BHDS.closeAllModals();

Hệ thống sự kiện

BHDS JS phát ra custom event trên từng component để bạn có thể lắng nghe và phản ứng.

Sự kiệnPhần tửDetail
bhds:tab:change[data-tabs]{ index, trigger, panel }
bhds:accordion:open[data-accordion-item]{ item, panel }
bhds:accordion:close[data-accordion-item]{ item, panel }
bhds:modal:open.modal-backdrop{ id }
bhds:modal:close.modal-backdrop{ id }
bhds:theme:changedocument{ theme: 'light' | 'dark' }
bhds:toast:showdocument{ message, type }
JavaScript
// Lắng nghe khi tab thay đổi
document.querySelector('[data-tabs]').addEventListener('bhds:tab:change', e => {
  console.log('Tab mới:', e.detail.index);
});

// Lắng nghe khi theme đổi
document.addEventListener('bhds:theme:change', e => {
  console.log('Theme hiện tại:', e.detail.theme);
});

// Lắng nghe khi modal mở
document.getElementById('myModal').addEventListener('bhds:modal:open', () => {
  // Tải dữ liệu khi modal mở
  fetchData();
});

Hệ thống Plugin

Mở rộng BHDS JS bằng cách đăng ký plugin. Plugin có thể thêm component mới, ghi đè hành vi, hoặc tích hợp thư viện bên ngoài.

JavaScript
// Tạo plugin
const MyPlugin = {
  name: 'my-plugin',
  install(BHDS) {
    // Thêm method mới
    BHDS.showConfirm = function(message, onConfirm) {
      // Logic hiển thị confirm dialog
    };

    // Lắng nghe sự kiện BHDS
    document.addEventListener('bhds:modal:open', e => {
      console.log('Modal mở:', e.detail.id);
    });
  }
};

// Đăng ký plugin
BHDS.use(MyPlugin);

// Khởi tạo sau khi tất cả plugin đã đăng ký
BHDS.init();

// Sử dụng method từ plugin
BHDS.showConfirm('Bạn có chắc?', () => deleteItem());
